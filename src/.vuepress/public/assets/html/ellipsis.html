多行文本省略号 (webkit方法)

限制显示行数，超出部分用省略号代替：

这是一段很长的文本内容，用来演示多行文本省略号的效果。当文本内容超过指定的行数时，多余的内容会被隐藏，并在最后一行的末尾显示省略号。这种方法使用了webkit的line-clamp属性，具有良好的浏览器支持度，是目前最推荐的多行省略号实现方案。 这是一段很长的文本内容，用来演示多行文本省略号的效果。当文本内容超过指定的行数时，多余的内容会被隐藏，并在最后一行的末尾显示省略号。这种方法使用了webkit的line-clamp属性，具有良好的浏览器支持度，是目前最推荐的多行省略号实现方案。
.multi-line-ellipsis { display: -webkit-box; /* 弹性伸缩盒子 */ -webkit-box-orient: vertical; /* 垂直排列 */ -webkit-line-clamp: 3; /* 限制为 3 行 */ overflow: hidden; /* 超出隐藏 */ text-overflow: ellipsis; /* 显示省略号 */ }

# UI 设计规范文档

## 一、边距与间距

**原则**：边距（左右）要大于间距（上下）。

### 1. 模块边距

* 常用：**15px**
* 可选：10px、12px、16px
* 最小：10px（用于需要展示更多内容的场景）

### 2. 模块间距

* 常用：10px、15px、20px
* 可选：8px、16px

### 3. 内容间距

* 常用：5px、10px、15px

### 4. 内容边距

* 常用：5px、10px、15px

### 5. 列表

* 最小高度：40px
* 单行高度：53px、55px、66px

## 二、全局规则

### 1. 图片比例

常用比例：

* 16:9
* 4:3
* 3:2
* 1:1
* 1:0.618（黄金比例）

### 2. 字体

**原则**：在统一的 UI 系统中建立字体体系，对不同类别（主、次、辅助、标题、展示）的字体进行规划，再在具体场景中微调。

* **字重**：

  * Regular (400)
  * Medium (500)
  * 英文加粗用 Semibold (600)

* **字号选择**：控制在 3–5 种，保持克制。

  * 14px：提示语、辅助文字
  * 16px：主文字
  * 18px：标题
  * 20px：大标题
  * > 20px：最大标题

* **行高**：推荐 **1.412 倍字号**

### 3. 颜色

* 确定主题色，定义字体、背景、禁用、边框线条、提示色（红橙黄绿蓝）。
* 每种色值建议提供 **2–3 个层级**（如文字颜色：主 `#333`、次 `#666`、提示 `#999`）。
* 常用色系：黑、白、主题色、绿（成功）、红（失败）、黄（警告）。

### 4. 阴影

**规律**：离地越远 → 阴影更淡、模糊度更高、面积更大；离地越近 → 阴影更深、模糊度更低、面积更小。

* **方向应用**：

  * 向下：组件内部或组件本身（常规场景）
  * 向上：底部导航、工具栏
  * 向左：右侧导航栏、抽屉组件、固定表格栏
  * 向右：左侧导航栏、抽屉组件、固定表格栏

* **分级阴影示例**：

  * **一级阴影**

    ```css
    0px 1px 2px -2px rgba(0, 0, 0, 0.16),
    0px 3px 6px 0px rgba(0, 0, 0, 0.12),
    0px 5px 12px 4px rgba(0, 0, 0, 0.09);
    ```
  * **二级阴影**

    ```css
    0px 3px 6px -4px rgba(0, 0, 0, 0.12),
    0px 6px 16px 0px rgba(0, 0, 0, 0.08),
    0px 9px 28px 8px rgba(0, 0, 0, 0.05);
    ```
  * **三级阴影**

    ```css
    0px 6px 16px -8px rgba(0, 0, 0, 0.08),
    0px 9px 28px 0px rgba(0, 0, 0, 0.05),
    0px 12px 48px 16px rgba(0, 0, 0, 0.03);
    ```
